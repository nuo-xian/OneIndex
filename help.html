OneIndex 帮助 chevron_right 获取应用ID和机密

获取应用ID和机密（E-mail：nuo_xian@qq.com）

1、点击安装界面的 获取应用ID和机密 来注册应用（需要登录微软账户，请使用需要使用OneIndex的微软账户）

2、点击左上角的分类，选择Azure Active Directory

3、然后点击应用注册，选择新注册

4、填写下图内容，并选择注册

5、点击注册后会跳转到以下页面，复制 应用程序(客户端) ID 的内容到安装界面的 应用 ID （应用程序(客户端) ID可以保存到本地，防止忘记）

6、点击 API 权限 > 添加权限，找到并选择 SharePoint 按照下图找到 MyFiles 选择 MyFiles.Read 和 MyFiles.Write 然后点击添加权限

7、点击 证书和密码 > 新客户端密码 按照下图输入内容并点击添加按钮，复制 值 的内容到安装界面的 应用机密（值只显示一次，可以保存到本地，防止忘记）

8、返回到安装界面，点击下一步。绑定需要使用的OneIndex微软账户即可安装成功
